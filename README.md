# ⭐ כוכב הכפל - משחק לוח הכפל לילדים

משחק אינטראקטיבי ומהנה ללימוד לוח הכפל, מותאם לילדי כיתה ב׳.

## 🦊 הכירו את כפלי!

כפלי הוא שועל חמוד שמלווה את הילדים לאורך המשחק, מעודד בתשובות נכונות ומחזק בטעויות.

## 🎮 מצבי משחק

### 🎯 תרגול חופשי
בחרו לוח כפל ספציפי (1-10) או ערבבו הכל. 10 שאלות עם 4 אפשרויות תשובה.

### ⏱️ אתגר הזמן
כמה תרגילי כפל תצליחו לפתור תוך 60 שניות? עם בונוס זמן ברצף של 5 תשובות נכונות!

### 🌟 מסע הכוכבים
11 שלבים מתקדמים - כל שלב מתמקד בלוח כפל אחר. אספו כוכבים ופתחו שלבים חדשים!

## ✨ פיצ׳רים

- 🦊 דמות מנחה "כפלי" מצוירת ב-CSS
- 🎯 כפל 1×1 עד 10×10 (תוצאה עד 100)
- 🎨 עיצוב צבעוני וידידותי לילדים
- 🎆 אפקטים: קונפטי, כוכבים מעופפים, אנימציות
- 🔥 מעקב רצף (streak) עם אפקט אש
- 💾 שמירת התקדמות ושיאים
- 📱 רספונסיבי - עובד על מחשב וטאבלט
- 🇮🇱 ממשק בעברית מלא

## 🚀 התקנה והרצה

פשוט פתחו את `index.html` בדפדפן - זה הכל! אין צורך בשרת או התקנה.

```bash
# או באמצעות שורת פקודה:
open index.html
```
